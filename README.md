# Airline Passenger Ticketing System

Developed an interactive airline ticketing system in Java, applying strict Object-Oriented Programming (OOP) principles. 

### Features
* Designed a robust class hierarchy using inheritance, polymorphism, and abstract classes to manage dynamic pricing and feature scaling across six distinct passenger tiers.
* Implemented real-time state tracking and input validation to render a dynamic seating map.
* Utilized Java Collections for efficient data management without third-party libraries.

### Technologies
* Java
* Object-Oriented Programming (OOP)
* Command-Line Interface (CLI)
